ISPS

Indonesian Stock Pattern Screener is a web application (WebApp) containing an interactive dashboard that visualizes stock data from Yahoo Finance as well as a screener for Indonesian stock patterns. ISPS aims to help investors understand stock chart and select potential stocks that match the investor's criteria.
